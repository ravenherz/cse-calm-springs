Database management system
rhz-we runs MongoDB by default. And I'm not quite sure that I will add support for something else except this particular DBMS
If Apache Tomcat 7 runs on the same machine as MongoDB does then just use localhost or 127.0.0.1 (no protocol)
By default, MongoDB runs at port 27017, but if your setup differs from the default one, please provide the correct one
Please specify the correct previously created database title, all collections needed by rhz-we will be located within this database
Privileged user credentials (DBMS and database access)
Needless to say, you should enable authentication on your mongo configuration. The user with the specified login has to be created before the next step
Please, avoid of using short insecure passwords
